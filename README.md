## 📂 Featured Projects

### 1️⃣ [AI Football Scout Predictor](./AI_Football_Scout_Predictor/)
- **Tech Stack:** Python, `pandas`, Local Llama 3.1, Parquet
- **Overview:** An advanced predictive scouting system that leverages Local LLMs and Data Engineering. It ingests match data from various APIs, calculates player performance using custom mathematical models (time decay, standard deviation, and favoritism weights), and outputs the statistically optimal lineup.
- **Business Value:** Showcases advanced data aggregation, statistical modeling, and local LLM decision-making capabilities.

### 2️⃣ [Algorithmic Crypto Trading Bot](./Algorithmic_Crypto_Trading_Bot/)
- **Tech Stack:** Python, `python-telegram-bot`, `ccxt`, REST APIs
- **Overview:** A robust automated trading system integrated with Binance and decentralized exchanges (Jupiter/Birdeye). It applies strict mathematical logic to calculate technical indicators (RSI, Moving Averages, Support/Resistance) in real-time and executes precise buy/sell signals via Telegram.
- **Business Value:** Demonstrates expertise in building fault-tolerant backend services, algorithmic trading, and real-time financial API integration.

### 3️⃣ [Multimodal AI Chart Analyzer](./Multimodal_AI_Chart_Analyzer/)
- **Tech Stack:** Python, Google Gemini Vision API, `Pillow`, `python-telegram-bot`
- **Overview:** A computer-vision-powered Telegram bot that uses multimodal AI to instantly analyze complex financial charts uploaded by users. It extracts visual patterns and cross-references them with real-time web search data to deliver highly accurate market sentiment analysis.
- **Business Value:** Highlights proficiency in cutting-edge Multimodal AI, Computer Vision pipelines, and intelligent AI Agents.

### 4️⃣ [Intelligent RPA: Cognitive Invoice Extractor](./Intelligent_RPA/)
- **Tech Stack:** Python, `PyPDF2`, `pandas`, Local Llama 3.1
- **Overview:** A zero-shot semantic extraction pipeline that parses raw text from PDF invoices and dynamically extracts structured JSON objects using a Local LLM, replacing fragile traditional RegEx.
- **Business Value:** Eradicates manual data entry operations in back-office financial workflows with 100% data privacy.

### 5️⃣ [AI Customer Service Agent: Private RAG Telegram Bot](./AI_Customer_Service_Bot/)
- **Tech Stack:** Python, `python-telegram-bot`, Offline LLM Inference
- **Overview:** An asynchronous Customer Support chatbot built for Telegram employing a local **RAG (Retrieval-Augmented Generation)** architecture to prevent LLM hallucination and ensure strict adherence to internal Knowledge Bases.
- **Business Value:** Provides scalable, 24/7 Level-1 customer support without leaking internal enterprise data.

### 6️⃣ [AI Data Scraper: Automated Market Intelligence](./AI_Data_Scraper/)
- **Tech Stack:** Python, `BeautifulSoup`, Playwright/Requests, LLM Prompt Engineering
- **Overview:** An autonomous web scraping loop that parses high-impact web headlines and forwards them to an AI Data Analyst to write real-time Markdown Executive Summaries.
- **Business Value:** Accelerates C-Level decision-making by distilling massive amounts of unstructured web noise into clean, actionable intelligence.
